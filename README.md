# Flipkart Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on laptop datasets using Python and Jupyter Notebook. The goal is to analyze laptop specifications, pricing, brands, ratings, and other features to gain meaningful insights.

## Files Included
- `flipkart.ipynb` - Jupyter Notebook containing the analysis.
- `laptops.csv` - Dataset used for analysis.
- `laptop_data.csv` - Additional dataset used in the project.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Statistical Insights

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Arun1859/Flipkart-Data-Analysis.git
   ```

2. Install required libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn jupyter
   ```

3. Open Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

4. Run `flipkart.ipynb`.
